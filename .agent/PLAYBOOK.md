# PLAYBOOK — Build & Operate Instructions

## Build Phase
1. Scaffold the project from the template
2. Implement core features based on business description
3. Add auth, payments, and user management
4. Write tests for critical paths
5. Deploy to staging, verify, then promote to production

## Operate Phase
1. Monitor error logs and user feedback
2. Fix bugs within the same session when possible
3. Propose feature improvements based on usage patterns
4. Keep dependencies updated

## Deployment Checklist
- [ ] All tests pass
- [ ] No TypeScript errors
- [ ] Environment variables set
- [ ] Database schema deployed
- [ ] Frontend builds successfully
- [ ] Smoke test after deploy
